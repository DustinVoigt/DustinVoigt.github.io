Video Narration

When the program opens you login with your Username and Account Number. After logging in you can choose to Withdraw money and the program will ask how much? Or you can choose to Deposit money and the program will ask you how much you would like to deposit. There is also option at the top right corner which will drop down with help or logout. If you choose to logout the program will log you out but if you choose help it will explain what are the options you can do. Next to options there is a more section which tell you about how the program was made and who made it.

Written Response

One difficulty we had trouble with was trying to figure out how to properly subtract and add the amount of money the user would like. The way we fixed this was by making balance a variable instead of setting a value to it. When you set balance as a variable you can subtract and add money without constantly resting the amount. Another problem we encountered with this project was trying to make the withdraw and deposit button disappear and reappear. The way we fixed this was by setting it invisible before the user was logged in and make in visible after the user logged in. This development was collaborated with me and my partner. A point that was independently done was making the dropdown menus. This was seen as an opportunity because before this project i had no clue how to make dropdown menus until this point.
