Video Narration

In the video it shows the running of the program and as you type menu() you can choose to Encrypt, Decrypt or to exit the program. When you Choose to encrypt, the program will ask for the “e” value and the “n” value. After that it will ask you what message you would like to encrypt and it will give you an encrypted message. The program can also decrypt by asking you for your “d” and for your “n” value. Then you will put in whatever encrypted code you are trying to decrypt and it will show you your decrypted message. Once you are done you can choose to exit the program and it will exit you out of the while loop.

Written Response

One difficulty with the program was getting the encryption code to work properly and have it ask for the “n” and the “e” value. An opportunity encountered was designing the menu portion so the user can select whether he/she wants the encrypt, decrypt or exit. Another opportunity was having other people help with the encryption so I could get a better understanding of the encryption process and where certain code should be. The encryption code was solved by having certain code lined up underneath each other to make sure some code was in the while loop and other code wasn't. I was able to create the menu function independently so when the user chooses to encrypt, it will take them to encrypt messages and the same if the user chooses to decrypt. The User also has an option to exit the program which will display a message letting them know they are done.
