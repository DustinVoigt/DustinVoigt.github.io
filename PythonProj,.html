Written Response

Our program was to create a hangman game where there computer chooses a word and the player has 6 lives to guess the word or else he/she loses. One difficulty I had was getting the program to keep track of the lives the players had. Everytime the player would lose a life the program would take away all the players lives thus causing the game to end on the first turn. The way we fixed it was by recoding the While loop but leaving the code to track the players life out of it. At one point I had the opportunity to understand and create a function that integrates another function. In hangman we had the secret (The word chosen by the program) call the while loop to start the game stating that the program is ready for the player to start guessing. The player lives counter was also subtracting double the amount of lives it was supposed to do. I was able to complete this independently by removing the code out of the while loop and replacing it below the while loop so it would get repeated twice or more times.
